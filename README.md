# WebDataScope
WebDataScope是一个正在打造的Chrome/Edge插件。它可以让你在平台上方便的获得其数据分析报告

使用方法和功能详见使用教程

# 存储的变量
- `WQPSettings`: 插件设置
- `WQPOPSAna`: Genius运算符使用分析结果
- `WQPRankData`: Genius排名分析结果
